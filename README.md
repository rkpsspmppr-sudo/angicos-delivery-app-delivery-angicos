# Angicos Delivery App

Aplicativo de delivery para pedidos de comida, restaurantes e entregas na cidade de Angicos/RN.

## 📱 Funcionalidades
- Cadastro de usuários
- Lista de restaurantes
- Carrinho de compras
- Acompanhamento de pedidos
- Área para entregadores

## 🚀 Objetivo
Facilitar pedidos de delivery na cidade de Angicos, conectando clientes, restaurantes e entregadores em um único app.

## 🛠️ Tecnologias (futuro)
- Frontend: React Native
- Backend: Node.js
- Banco de dados: Firebase ou MongoDB

## 📍 Localização
Angicos - Rio Grande do Norte
